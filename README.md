API desenvolvida para resolver a dificuldade de coordenar o uso de salas e assentos em empresas no modelo híbrido. Permite consultar disponibilidade de espaços por dia, horário, lotação, local e tipo, realizar reservas e conta com IA para recomendar a melhor opção conforme a necessidade do usuário.
Funcionalidades

Cadastro de usuários
Consulta de salas por dia, horário, lotação, local, tipo e disponibilidade
Reserva de salas e assentos
Recomendação inteligente de espaço com base nos critérios do usuário
Extração automática de layout a partir do envio da planta da sala
